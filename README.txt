FIXED v2 사용법

1. GitHub에는 index.html만 올립니다.
2. Google Sheets > 확장 프로그램 > Apps Script에는 Code.gs만 붙여넣습니다.
3. Code.gs 맨 위 SERVER_SECRET을 새 비밀번호로 바꿉니다.
4. Apps Script 저장 후 배포 > 새 배포 > 웹 앱으로 배포합니다.
5. GitHub Pages 화면에서 웹 앱 URL과 비밀번호를 넣고 연결 테스트합니다.

index.html 화면 제목이 "CS 문의 관리 - FIXED v2"로 보여야 새 파일이 반영된 것입니다.
