CS 문의 관리 & 자동답변 도구 - 실사용 버전

1. 실행 방법
- 압축을 풀고 index.html을 더블클릭하세요.
- 설치 없이 크롬/엣지 브라우저에서 바로 실행됩니다.

2. 주요 기능
- 문의 내용을 직접 입력할 수 있습니다.
- 입력한 문의를 자동 분류합니다.
- 답변 초안을 자동 생성합니다.
- 문의를 저장하고 미처리/처리중/완료 상태로 관리할 수 있습니다.
- 메모를 남길 수 있습니다.
- 분류 키워드와 답변 템플릿을 직접 수정할 수 있습니다.
- CSV 다운로드가 가능합니다.
- JSON 백업/복구가 가능합니다.
- 파일을 열면 하루 1번 자가 테스트가 자동 실행됩니다.

3. 데이터 저장 방식
- 입력한 문의와 수정한 규칙은 현재 사용하는 브라우저의 localStorage에 저장됩니다.
- 같은 PC/같은 브라우저에서 다시 열면 유지됩니다.
- 다른 PC에서 사용하려면 '자가 테스트/백업' 탭에서 전체 백업을 다운로드하고, 새 PC에서 백업을 불러오세요.

4. 주의
- 이 버전은 서버가 없는 로컬 파일 방식입니다.
- 브라우저 캐시/사이트 데이터 삭제 시 저장 내용이 사라질 수 있으므로 주기적으로 백업하세요.
- ChatGPT가 사용자의 PC 파일을 매일 자동으로 열어 직접 테스트하는 방식은 불가능합니다.
  대신 프로그램 내부에 '매일 자가 테스트' 기능을 넣었습니다.
